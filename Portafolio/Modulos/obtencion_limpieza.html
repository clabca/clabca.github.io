Módulo Obtención y Limpieza de Datos

Este módulo presenta mis habilidades para extraer, transformar y preparar datos de diversas fuentes, asegurando que la información esté limpia, consistente y lista para análisis avanzados o modelamiento.

Habilidades en Obtención de Datos

  • Extracción de datos desde bases de datos SQL y NoSQL.
  • Obtención de información desde APIs, archivos CSV, JSON y Excel.
  • Web scraping y automatización de descarga de datos públicos.
  • Conexión a servicios de almacenamiento en la nube para ingestión de datos.
  • Uso de Python y librerías como Pandas, Requests y BeautifulSoup.

Habilidades en Limpieza de Datos

  • Detección y tratamiento de valores nulos, duplicados o inconsistentes.
  • Normalización y estandarización de formatos de datos.
  • Transformaciones y creación de variables derivadas para análisis posteriores.
  • Validación de integridad y consistencia de datasets.
  • Optimización de grandes datasets para procesamiento eficiente.

Proyectos Destacados

  • Pipeline de limpieza y preparación de datos crudos de transporte aéreo para análisis de demanda.
  • Automatización de extracción de información de portales públicos mediante web scraping.
  • Normalización de datasets heterogéneos y combinación para reportes consolidados.
  • Validación de calidad de datos mediante scripts automatizados en Python.

Herramientas y Librerías

  • Python: Pandas, NumPy, Requests, BeautifulSoup
  • SQL y herramientas de consultas avanzadas
  • Jupyter Notebooks
  • Git y GitHub
  • Servicios en la nube (AWS S3, GCP Storage)
