Módulo Big Data

En este módulo muestro mis competencias en el manejo de grandes volúmenes de datos, aplicando tecnologías y arquitecturas distribuidas para el almacenamiento, procesamiento y análisis eficiente de información masiva.

Habilidades en Big Data

  • Procesamiento de grandes volúmenes de datos en entornos distribuidos.
  • Diseño de arquitecturas de Big Data escalables y seguras.
  • Ingesta y transformación de datos en tiempo real y batch.
  • Optimización de pipelines de datos para mejorar rendimiento.
  • Integración de datos desde múltiples fuentes heterogéneas.

Proyectos Destacados

  • Procesamiento de datasets masivos de transporte aéreo con Apache Spark.
  • Diseño de pipelines en Apache NiFi para ingesta de datos en tiempo real.
  • Implementación de almacenamiento distribuido en HDFS.
  • Optimización de consultas sobre grandes volúmenes de datos en clusters.

Herramientas y Tecnologías

  • Apache Spark, Hadoop, Kafka
  • Apache NiFi para orquestación de flujos
  • Almacenamiento distribuido: HDFS, S3
  • Lenguajes: Python, SQL
  • Contenedores y clusters: Docker, Kubernetes
